## ⚡ 快速部署（3步完成）

### 第 1 步：准备代码

确保代码已推送到 GitHub（如果还没有）：

```bash
git init
git add .
git commit -m "准备部署"
git remote add origin https://github.com/你的用户名/你的仓库名.git
git branch -M main
git push -u origin main
```

### 第 2 步：部署机器人 1

1. **访问 Railway**
   - 打开 [railway.app](https://railway.app)
   - 用 GitHub 登录

2. **创建项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你的代码仓库

3. **配置环境变量**
   - 点击 "Variables"
   - 添加：
     ```
     BOT_TOKEN=8297508981:AAGnMA5T6ltbzHU4LvWcG6of5HxYj8h1yT4
     ```

4. **完成！** Railway 会自动部署

### 第 3 步：部署机器人 2

1. **创建新项目**
   - 在 Railway 中点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - **选择同一个代码仓库**

2. **配置环境变量**
   - 点击 "Variables"

## 🎯 重要提示

### 1. 两个机器人可以共用一个仓库

- ✅ 使用相同的代码仓库
- ✅ 分别创建两个 Railway 项目
- ✅ 设置不同的 `BOT_TOKEN` 环境变量

### 2. 互不影响

- ✅ 每个机器人独立运行
- ✅ 独立日志
- ✅ 独立管理

### 3. 给机器人管理员权限

部署完成后，在群组中：
- ✅ 给机器人 1 管理员权限
- ✅ 给机器人 2 管理员权限
- ✅ 权限：删除消息、限制成员、封禁成员

## 🐛 遇到问题？

### 机器人不响应

1. 检查 `BOT_TOKEN` 是否正确
2. 查看 Railway 日志是否有错误
3. 确认机器人已启动

### 部署失败

1. 查看部署日志中的错误
2. 确认代码已推送到 GitHub
3. 确认 `requirements.txt` 存在
