## 🚀 Railway 快速部署步骤

### 部署第一个机器人

1. **访问 Railway**
   - 打开 [railway.app](https://railway.app)
   - 使用 GitHub 账号登录

2. **创建项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你的代码仓库
   - 点击 "Deploy Now"

3. **配置环境变量**
   - 在项目页面，点击 "Variables" 标签
   - 点击 "+ New Variable"
   - 添加：
     - **名称**：`BOT_TOKEN`
     - **值**：`8297508981:AAGnMA5T6ltbzHU4LvWcG6of5HxYj8h1yT4`
   - 点击 "Add" 保存

4. **等待部署**
   - Railway 会自动安装依赖并启动
   - 查看日志：点击部署卡片 → "View Logs"
   - 看到 "机器人启动中..." 说明成功

5. **测试机器人**
   - 在 Telegram 中发送 `/start`
   - 如果收到回复，部署成功！

## 🐛 常见问题

### Q1: 两个机器人可以共用一个仓库吗？

**A**: 可以！Railway 支持同一个仓库部署多个项目，只需要设置不同的环境变量即可。

### Q2: 两个机器人会互相影响吗？

**A**: 不会。每个 Railway 项目都是独立的，使用不同的 `BOT_TOKEN`，互不影响。

### Q3: 如何区分两个机器人？

**A**: 
- 在 Railway 中给项目命名（如 "机器人1"、"机器人2"）
- 在 Telegram 中通过机器人用户名区分

### Q4: 如何查看两个机器人的日志？

**A**: 
- 在 Railway 项目列表中
- 分别点击每个项目
- 查看各自的日志
