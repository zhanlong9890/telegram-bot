# 机器人功能规划

## 📋 当前已有功能

- ✅ 踢出用户 (`/ban`)
- ✅ 解封用户 (`/unban`)
- ✅ 禁言用户 (`/mute`)
- ✅ 解除禁言 (`/unmute`)
- ✅ 警告用户 (`/warn`) - ⚠️ **需要完善：添加警告计数和数据库存储**
- ✅ 删除消息 (`/del`)
- ✅ 查看用户信息 (`/info`)

## 🚀 建议添加的功能

### 1. 警告系统完善（重要）
- [ ] 添加数据库存储警告记录
- [ ] 警告计数功能（/warns 查看用户警告次数）
- [ ] 达到警告上限自动踢出
- [ ] 清除警告功能 (`/unwarn`)

### 2. 自动管理功能
- [ ] 自动删除广告链接
- [ ] 自动删除特定关键词
- [ ] 新成员欢迎消息
- [ ] 自动踢出新加入的机器人
- [ ] 限制新成员发言（验证期）

### 3. 群组设置功能
- [ ] 设置欢迎消息 (`/setwelcome`)
- [ ] 设置规则 (`/setrules`)
- [ ] 查看群组统计 (`/stats`)
- [ ] 设置自动删除时间

### 4. 高级管理功能
- [ ] 批量操作（批量踢出、禁言）
- [ ] 黑名单管理 (`/blacklist`)
- [ ] 白名单管理 (`/whitelist`)
- [ ] 管理员列表 (`/admins`)
- [ ] 群组信息 (`/groupinfo`)

### 5. 反垃圾功能
- [ ] 检测并删除重复消息
- [ ] 限制消息频率（防刷屏）
- [ ] 检测并删除长文本（可能是垃圾）
- [ ] 检测并删除特定格式消息

### 6. 实用工具
- [ ] 获取用户ID (`/id`)
- [ ] 获取群组ID (`/chatid`)
- [ ] 消息转发功能
- [ ] 定时消息

### 7. 数据存储
- [ ] SQLite 数据库存储
- [ ] 警告记录
- [ ] 黑名单
- [ ] 群组设置
- [ ] 操作日志

## 🎯 优先级建议

### 高优先级（核心功能）
1. **警告系统完善** - 添加数据库和计数功能
2. **自动删除广告** - 自动管理的基础
3. **新成员欢迎** - 提升用户体验
4. **群组设置** - 让每个群组可以自定义

### 中优先级（实用功能）
5. 黑名单管理
6. 群组统计
7. 反垃圾功能

### 低优先级（高级功能）
8. 批量操作
9. 定时消息
10. 消息备份
