  Telegram 机器人云端部署指南（Railway - 推荐）
═══════════════════════════════════════════════════════════

【第一步】准备代码
─────────────────────────────────────────────────────────
1. 确保代码已推送到 GitHub
   - 如果没有，先创建 GitHub 仓库
   - 推送代码到仓库

【第二步】注册 Railway
─────────────────────────────────────────────────────────
1. 访问：https://railway.app
2. 点击 "Login" → 选择 "Login with GitHub"
3. 授权 Railway 访问你的 GitHub

【第三步】创建项目
─────────────────────────────────────────────────────────
1. 点击 "New Project"
2. 选择 "Deploy from GitHub repo"
3. 选择你的代码仓库
4. 点击 "Deploy Now"

【第四步】配置环境变量
─────────────────────────────────────────────────────────
1. 在项目页面，点击 "Variables" 标签
2. 点击 "+ New Variable"
3. 添加变量：
   名称：BOT_TOKEN
   值：你的机器人Token（从 @BotFather 获取）
4. 点击 "Add" 保存

【第五步】等待部署
─────────────────────────────────────────────────────────
- Railway 会自动安装依赖并启动机器人
- 查看日志：点击部署卡片 → "View Logs"
- 看到 "机器人启动中..." 说明成功

【第六步】测试
─────────────────────────────────────────────────────────
1. 在 Telegram 中找到你的机器人
2. 发送 /start 命令
3. 如果收到回复，部署成功！
